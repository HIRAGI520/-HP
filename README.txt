【このサイトの使い方】

1. まずは site-data.js を開いて、塾名・料金・連絡先などを書き換えてください。
2. index.html をブラウザで開くと、サイトを確認できます。
3. デザインを変えたいときは style.css を編集してください。

【一番かんたんな編集ポイント】
- 塾名: site-data.js の schoolName
- タイトル文: heroTitle
- 料金: priceValue, priceNote
- LINE: lineUrl
- メール: mailAddress

【公開するには】
- GitHub Pages
- Netlify
- Vercel
などにアップロードすれば無料で公開できます。

【注意】
- Google Fonts を読み込んでいるので、ネット接続があると見た目が安定します。
- もっと本格的にしたい場合は、写真、合格実績、講師紹介、LINEボタン固定などを追加できます。
